# Audio assets

The WAV effects in this folder were procedurally generated for this prototype using a small Python script in the repository.
They are released into the public domain (CC0) so they can be used freely in the game.

| File | Purpose |
| --- | --- |
| `jump.wav` | Jump and landing chirp |
| `sword.wav` | Sword swing whoosh |
| `hurt.wav` | Player damage buzz |
| `enemy_hit.wav` | Enemy impact thump |
| `enemy_defeat.wav` | Enemy defeat sparkle |
| `coin.wav` | Coin pickup ping |
| `block.wav` | Shield block tick |
| `shop.wav` | Shop purchase chime |
